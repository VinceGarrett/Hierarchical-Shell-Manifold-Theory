# Mathematica Notebooks for Hierarchical Shell-Manifold Theory (HSMT)

This folder contains all symbolic derivations, proofs, and numerical scripts referenced in the HSMT paper (February 28, 2026 version).

## Files

- **HSMT_Core_Derivations.nb**  
  The main notebook containing the complete symbolic derivations for the entire theory.

- **SM_Overlap_Integrals.nb**  
  Explicit calculation of all 19 Standard Model parameters from lower-shell wavefunction overlaps.

- **Functional_Variation.nb**  
  Full variation of the refined action with respect to the metric and the blurriness field σ(r, t).

- **No_Signalling_Proof.nb**  
  Operator-algebra proof of the no-signalling theorem for resonant leakage channels.

- **Zeta_Function_UV_Completion.nb**  
  Exact zeta-function regularisation and non-perturbative ultraviolet completion.

## How to Use
Open any notebook in Mathematica (version 12 or later recommended). All notebooks use only built-in functions (no external packages required). The main notebook `HSMT_Core_Derivations.nb` is organised by paper section numbers for easy reference.

All results match the equations and tables in the paper exactly. The notebooks are provided as both symbolic verification and reproducible supplementary material.
