# Global MCMC Fits

This folder contains the full Markov-chain Monte Carlo analysis used for the cosmological validation in the HSMT paper.

Contents:
- Joint fit to Planck 2018 CMB, DESI 2024 BAO, Pantheon+ supernovae, SH0ES H₀, and BBN abundances
- Best-fit parameters (H₀ = 70.2 ± 1.1 km s⁻¹ Mpc⁻¹, Ω_m h² = 0.143 ± 0.002, etc.)
- Resolution of H₀ tension (~1.6σ) and lithium problem
- Chain files, corner plots, and χ² comparison with ΛCDM

Scripts use the modified CLASS code with projection kernel and leakage terms.
